# Cadastro de Nomes em Java

Este é o meu primeiro projeto em Java. É um projeto simples, mas foi desenvolvido com muita dedicação e paciência durante meu processo de aprendizado.

## Objetivo

O objetivo deste projeto foi praticar conceitos fundamentais da linguagem Java, como:

- ArrayList
- Scanner
- Métodos `static`
- Variáveis
- Métodos
- Estruturas condicionais (`if/else`)
- Estruturas de repetição (`while` e `for`)
- Entrada e saída de dados (`Scanner` e `System.out.println`)
- Encerramento de recursos com `close()`
- Percorrer listas utilizando laços de repetição

## Boas práticas

Neste projeto procurei aplicar um princípio de **Clean Code** conhecido como **DRY (Don't Repeat Yourself)**, criando um método para exibir o menu e evitando repetir o mesmo trecho de código.

## Tecnologias

- Java
- JDK 21 (ou a versão que você estiver utilizando)
